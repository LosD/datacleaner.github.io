Density plot

Plots the occurences of two number variables in a density plot chart. A useful visualization for identifying freqencies of combinations in numeric data relationships.

Analyzer

Result metrics (IDensityAnalyzerResult)

Configured properties

  • Variable1

    The field with the first variable. Will be plotted on the horizontal X-axis.

    InputColumn<Number> Required

  • Variable2

    The field with the second variable. Will be plotted on the vertical Y-axis.

    InputColumn<Number> Required

Round number

Transformation for rounding a number, typically to the nearest integer, nearest ten, hundred or thousand.

Transformer Concurrent Distributed execution possible

Configured properties

  • Number

    InputColumn<Number> Required

  • Round factor

    Defines the factor of rounding. A factor of 1 will round to the nearest integer. A factor of 1000 will round the number to the nearest thousand.

    int Required
